# Lombriweb 🌿✨

Lombriweb es una plataforma digital enfocada en la comercialización de productos orgánicos, diseñando una experiencia directa y funcional para el usuario. 

## Sobre el proyecto
Este proyecto nace de la necesidad de conectar a los productores orgánicos con sus clientes de forma inmediata, eliminando intermediarios innecesarios. Lombriweb permite explorar un catálogo seleccionado de humus y fertilizantes orgánicos, facilitando la compra directa a través de WhatsApp.

## Características
* **Catálogo dinámico**: Visualización clara de productos con precios y descripción.
* **Integración con WhatsApp**: Pedidos rápidos y directos mediante un solo clic.
* **Diseño responsivo**: Adaptado para una navegación fluida en dispositivos móviles.

## Tecnologías utilizadas
* **HTML5** y **CSS3**: Para una estructura limpia y un estilo visual atractivo.
* **JavaScript**: Para la lógica de carga de productos y gestión de pedidos.
* **GitHub Pages**: Para el despliegue y puesta en marcha del proyecto.
